2

Configure GitHub webhook

{% if config.github_webhook_secret %} ✓ Secret configured {% endif %}

In your GitHub repo go to Settings → Webhooks → Add webhook and fill in:

Payload URL
{{ webhook_url }}
Content type application/json
Events Pull requests
Secret a random string you generate (see below)

Generate a secure secret with:

openssl rand -hex 32

Set the same value as GITHUB_WEBHOOK_SECRET in your deployment environment.
